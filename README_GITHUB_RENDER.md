# Football Academy Manager Pro — GitHub + Render

Bản triển khai online gồm đầy đủ module:

- Admin: học viên, lớp, HLV, phụ huynh, lịch tập, điểm danh, học phí, QR, kho đồ, báo cáo, chatbot, Zalo OA.
- HLV: lớp được gán, lịch tập, điểm danh và theo dõi học viên.
- Phụ huynh: xem quá trình học của con, tải thẻ, xem công nợ và QR đóng học phí.

## Deploy nhanh

1. Upload toàn bộ source lên GitHub.
2. Trên Render chọn **New → Blueprint**.
3. Chọn repository; Render đọc `render.yaml` để tạo web + PostgreSQL.
4. Nhập tài khoản Admin lúc Render hỏi biến môi trường.
5. Mở link `.onrender.com` được cấp.

Xem hướng dẫn tiếng Việt đầy đủ tại: `HUONG_DAN_GITHUB_RENDER.md`.
